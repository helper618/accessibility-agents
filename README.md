## 📚 About accessibility-agents

Accessibility-agents helps you find and fix accessibility problems in code generated by AI tools like Claude Code, GitHub Copilot, and Claude Desktop. It uses eleven specialist reviewers to check if your code meets WCAG 2.2 AA standards. These standards ensure your websites and apps work well for everyone, including people with disabilities.

This tool is designed for users who want to improve the accessibility of AI-generated code. You do not need programming experience to use it. The application offers a straightforward way to scan your code and get clear feedback.

## 🛠 How to Use accessibility-agents

Once you open the app, you can start reviewing your code for accessibility. Here’s how:

1. **Open Your Code File**
   - Click the “Open File” button.
   - Navigate to your project files and select the code you want to check (HTML, JavaScript, or similar).

2. **Run the Accessibility Review**
   - Click “Start Review”.
   - The tool runs all eleven accessibility checks on your file.
   - This process usually takes a few seconds.

3. **View the Results**
   - The app displays a list of issues found.
   - Each issue will have an explanation and a suggestion to fix it.

4. **Fix the Problems**
   - Use the suggestions to edit your code.
   - You can rerun the review to check your changes.
